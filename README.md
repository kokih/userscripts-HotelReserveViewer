# userscripts-HotelReserveViewer

α-1 予約画面の表示改善スクリプト for Chrome TamperMonkey

- Version 1.2.0
  - 機能4: キャンセル済み予約の表示・非表示切替機能追加
- Version 1.1.0
  - 機能3: 喫煙・禁煙の背景色変更機能追加
  - 機能2: 出雲・鯖江の背景色変更機能追加
- Version 1.0.0
  - 機能1: 予約確認画面の、表ソート機能追加

## 使用方法

### Chome ブラウザを使用している場合

1. [拡張機能](https://chrome.google.com/webstore/category/extensions)画面を開き、Tampermonkey を検索して下さい
1. [TamperMonkey](https://www.tampermonkey.net/) を導入して下さい
1. 本サイトから、HotelReserveViewer をダウンロードして下さい
1. TamperMonkey にスクリプトを導入して下さい
