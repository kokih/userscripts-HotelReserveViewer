# userscripts-HotelReserveViewer
α-1 予約画面の表示改善スクリプト for Chrome TemperMonkey

Version 1.2.0 キャンセル済み予約の表示・非表示切替機能追加
Version 1.1.0 出雲・鯖江の背景色変更、喫煙・禁煙の背景色変更追加
Version 1.0.0 予約確認画面の、表ソート機能追加

## 使用方法

### Chome ブラウザを使用している場合

1. 拡張機能 TemperMonkey を導入して下さい
1. 本サイトから、HotelReserveViewer をダウンロードして下さい
1. TemperMonkey にスクリプトを導入して下さい
